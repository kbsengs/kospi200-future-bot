## 핵심 인사이트

### 최적 파라미터 (1위)
```yaml
ema_length:    200
mom_lookback:  3
stop_atr_mult: 3.0
총손익: 44,535,000원 | PF: 1.76 | 승률: 52.5% | MDD: -4,310,000원
```

### 현재 config (13위)
```yaml
ema_length:    100   # 최적과 동일 효과 (1~3위 EMA 차이 미미)
mom_lookback:  5     # ★ 문제: 3이 최적
stop_atr_mult: 3.0   # 최적
총손익: 38,865,000원 | PF: 1.65 | 승률: 51.2%
```

### 파라미터별 영향도 분석

**mom_lookback이 가장 중요한 파라미터**
- lookback=3 : 평균 총손익 ~42,000,000원
- lookback=5 : 평균 총손익 ~37,000,000원
- lookback=8 : 평균 총손익 ~34,000,000원
- lookback=10: 평균 총손익 ~28,000,000원
→ **lookback이 짧을수록 수익 증가** (직전 구간만 참조해 진입 민감도 높임)

**stop_atr_mult: 3.0이 일관되게 최상위**
- stop=3.0이 같은 lookback에서 항상 1위
- 손절 범위가 넓을수록 성과 양호 (조기 손절 방지)

**EMA length: 차이 미미**
- 50/100/200 간 손익 차이 ~200,000원 수준
- 사실상 EMA 필터는 성과에 큰 영향 없음

### 권장 config 변경
```yaml
# 현재 → 권장 (1위 기준)
ema_length:    100 → 200  # 효과 미미, 변경 선택사항
mom_lookback:  5   → 3    # ★ 중요: 약 5,600,000원 손익 차이
stop_atr_mult: 3.0 → 3.0  # 유지
```
